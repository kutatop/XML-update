# Feed Filter

Автоматически фильтрует товарный фид `https://knigovan.com/price/exportPrice.xml`,
исключая офферы из определённых категорий (и всех их подкатегорий),
и сохраняет результат в `output/filtered_feed.xml`.

Результат обновляется автоматически раз в день через GitHub Actions
(см. `.github/workflows/update-feed.yml`).

## Прямая ссылка на актуальный файл

После того как первый запуск Actions завершится, файл будет доступен по адресу:

```
https://raw.githubusercontent.com/<ваш-логин>/<название-репозитория>/main/output/filtered_feed.xml
```

Замените `<ваш-логин>` и `<название-репозитория>` на реальные значения.

## Запуск вручную (без ожидания расписания)

1. Откройте вкладку **Actions** в репозитории на GitHub
2. Выберите workflow **Update filtered feed**
3. Нажмите **Run workflow**

## Изменить список исключаемых категорий

Откройте `filter_feed.py`, отредактируйте множество `EXCLUDED_CATEGORIES`
(список ID категорий — строки), сохраните и закоммитьте изменения.
При следующем запуске (по расписанию или вручную) фильтр применится
с новыми настройками.

## Изменить расписание

В файле `.github/workflows/update-feed.yml` строка:

```yaml
- cron: "0 3 * * *"
```

означает "каждый день в 03:00 UTC". Формат: `минута час день месяц день_недели`.
Примеры:
- `"0 */6 * * *"` — каждые 6 часов
- `"0 6,18 * * *"` — дважды в день, в 06:00 и 18:00 UTC

## Запуск локально (для проверки)

```bash
pip install -r requirements.txt  # зависимостей нет, скрипт использует только стандартную библиотеку
python filter_feed.py
```
